## 🔧 COMPONENTES CREADOS

### 1️⃣ INTERFAZ WEB (INTERFAZ_DE_CONTROL.php)

**Características:**
- ✅ Diseño moderno con colores PRERMI (gradientes azul-violeta)
- ✅ Controles de movimiento: botones direccionales + joystick virtual
- ✅ Control de velocidad individual por rueda (sliders 0-255)
- ✅ Control de 2 servomotores con botones + ángulos manuales
- ✅ Botón de emergencia prominente (rojo)
- ✅ Monitor de sistema en tiempo real
- ✅ Log de eventos con colores (success, error, info)
- ✅ Indicador de conexión (conectado/desconectado)
- ✅ Responsive (funciona en móvil)

**Ruedas Controladas:**
```
IN1 ← Rueda Delantera Izquierda     IN2 ← Rueda Delantera Derecha
IN3 ← Rueda Trasera Izquierda       IN4 ← Rueda Trasera Derecha
```

**Servomotores Controlados:**
```
ServoPin1 (D4) ← Levantador de Contenedor 1 (0-180°)
ServoPin2 (D3) ← Levantador de Contenedor 2 (0-180°)
```

---

### 2️⃣ API PHP (control_api.php)

**Características:**
- ✅ Recibe solicitudes POST con comandos en JSON
- ✅ Envía comandos al ESP8266 vía HTTP
- ✅ Registra todos los eventos en `control_log.txt`
- ✅ Maneja CORS para solicitudes cross-origin
- ✅ Validación de JSON
- ✅ Timeout automático (5 segundos)

**Comandos Soportados:**
| Comando | Acción |
|---------|--------|
| FORWARD | Todas las ruedas avanzan |
| BACKWARD | Todas las ruedas retroceden |
| LEFT | Giro a la izquierda |
| RIGHT | Giro a la derecha |
| STOP | Detener todos motores |
| SERVO1_MOVE | Mover servo1 a ángulo especificado |
| SERVO2_MOVE | Mover servo2 a ángulo especificado |
| EMERGENCY_STOP | Parada de emergencia (todos detienen) |

---

### 3️⃣ CÓDIGO ESP8266 (main.cpp)

**Características:**
- ✅ 400+ líneas de código
- ✅ **CADA LÍNEA tiene comentarios detallados** explicando QUÉ HACE
- ✅ Servidor web incorporado en el ESP8266 (puerto 80)
- ✅ Conexión automática a WiFi
- ✅ Parser JSON automático de comandos
- ✅ Control de 4 motores DC con PWM
- ✅ Control de 2 servomotores
- ✅ Seguridad: timeout de 5 segundos
- ✅ Monitoreo por Serial (115200 baud)

**Qué Hace:**
1. Conecta a tu red WiFi (SSID + Password)
2. Crea un servidor web en puerto 80
3. Escucha solicitudes HTTP POST en `/control`
4. Parsea JSON con comandos
5. Controla motores digitalmente (HIGH/LOW)
6. Controla velocidad con PWM (0-255)
7. Posiciona servos a ángulos específicos (0-180°)
8. Valida datos y responde al servidor XAMPP
