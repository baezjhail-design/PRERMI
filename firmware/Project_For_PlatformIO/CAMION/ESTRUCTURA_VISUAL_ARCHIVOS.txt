## 🔌 CONEXIONES FÍSICAS RESUMIDAS

### Motores → ESP8266:
```
L298N IN1 ←→ D2 (Rueda Frontal Izquierda)
L298N IN2 ←→ D1 (Rueda Frontal Derecha)
L298N IN3 ←→ D7 (Rueda Trasera Izquierda)
L298N IN4 ←→ D8 (Rueda Trasera Derecha)
L298N GND ←→ GND (IMPORTANTE: GND COMÚN)
```

### Servos → ESP8266:
```
Servo1 Señal ←→ D4 (Levantador 1)
Servo2 Señal ←→ D3 (Levantador 2)
```

### Alimentación:
```
Batería 12V+ ←→ L298N 12VIN
Batería GND  ←→ L298N GND (MISMO GND del ESP8266)
Regulador 5V ←→ ESP8266 VIN
```

## ⚡ COMANDOS DISPONIBLES

Desde la interfaz puedes enviar:

```
FORWARD          → Camión avanza
BACKWARD         → Camión retrocede
LEFT             → Gira a izquierda
RIGHT            → Gira a derecha
STOP             → Detiene motores
SERVO1_MOVE      → Posiciona servo1 (0-180°)
SERVO2_MOVE      → Posiciona servo2 (0-180°)
EMERGENCY_STOP   → Parada de emergencia
```

## 🛠️ MODIFICACIONES COMUNES (CHEAT SHEET)

### Cambiar WiFi:
```cpp
main.cpp línea ~20
const char* WIFI_SSID = "Tu-Red";
const char* WIFI_PASS = "Tu-Contraseña";
```

### Cambiar Config PHP:
```php
config.php línea ~9
define('ESP8266_IP', '10.0.0.123');  // Tu IP
```

### Agregar Comando:
```
1. En main.cpp: agregar else if en processCommand()
2. En control_api.php: agregar case en switch()
3. En INTERFAZ_DE_CONTROL.php: agregar HTML + botón
```

```cpp
main.cpp línea ~40
#define IN1 D5   // Cambiar D2 a D5
```
