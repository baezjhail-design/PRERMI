FLUJO 1: REGISTRO DE USUARIO (v1.0 - sin cambios)

FLUJO 2: REGISTRO DE ADMIN (v2.0 - NUEVO)
Admin Registers 
  → Email Sent + Token
    → Admin verifies email
      → Status: verified=1, active=0
        → Superadmin approves
          → Status: active=1
            → Admin can login ✓

FLUJO 3: VALIDACIÓN RFID (v2.0 - NUEVO)
User scans token on RFID reader
  → ESP32 reads token
    → POST to /api/contenedores/validar_token_rfid.php
      → Server validates in DB
        → Response: success/error
          → ESP32: LED+Buzzer reaction
            → Log event in logs_sistema ✓
